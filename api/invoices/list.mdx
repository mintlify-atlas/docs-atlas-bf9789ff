---
title: "List Invoices"
api: "GET /v1/invoices"
description: "Use when listing or searching invoices (e.g. admin view or customer history). Returns a paginated list; supports filtering by customer, status, date range."
---

## Query Parameters

<ParamField query="customer_id" type="string">
  Filter invoices by customer ID
</ParamField>

<ParamField query="subscription_id" type="string">
  Filter invoices by subscription ID
</ParamField>

<ParamField query="invoice_status" type="string">
  Filter by invoice status
  
  **Values:** `DRAFT`, `FINALIZED`, `VOIDED`, `UNCOLLECTIBLE`
</ParamField>

<ParamField query="payment_status" type="string">
  Filter by payment status
  
  **Values:** `PENDING`, `PAID`, `UNPAID`, `PARTIALLY_PAID`, `OVERPAID`, `FAILED`
</ParamField>

<ParamField query="invoice_type" type="string">
  Filter by invoice type
  
  **Values:** `SUBSCRIPTION`, `ONE_TIME`, `CREDIT`, `PRORATION`
</ParamField>

<ParamField query="due_date_from" type="string">
  Filter invoices with due date on or after this date (ISO 8601 format)
</ParamField>

<ParamField query="due_date_to" type="string">
  Filter invoices with due date on or before this date (ISO 8601 format)
</ParamField>

<ParamField query="period_start_from" type="string">
  Filter invoices with billing period starting on or after this date (ISO 8601 format)
</ParamField>

<ParamField query="period_start_to" type="string">
  Filter invoices with billing period starting on or before this date (ISO 8601 format)
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Number of invoices to return per page (max 100)
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Number of invoices to skip for pagination
</ParamField>

<ParamField query="sort_by" type="string" default="created_at">
  Field to sort by
  
  **Values:** `created_at`, `updated_at`, `due_date`, `period_start`, `total`
</ParamField>

<ParamField query="sort_order" type="string" default="desc">
  Sort order
  
  **Values:** `asc`, `desc`
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.flexprice.io/v1/invoices?customer_id=cust_1234&invoice_status=FINALIZED&limit=20" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
client = Flexprice(api_key="YOUR_API_KEY")

invoices = client.invoices.list(
    customer_id="cust_1234",
    invoice_status="FINALIZED",
    limit=20
)
```

```typescript TypeScript
const client = new Flexprice({ apiKey: "YOUR_API_KEY" });

const invoices = await client.invoices.list({
  customerId: "cust_1234",
  invoiceStatus: "FINALIZED",
  limit: 20
});
```

```go Go
client := flexprice.NewClient("YOUR_API_KEY")

invoices, err := client.Invoices.List(context.Background(), &flexprice.InvoiceListParams{
    CustomerID: "cust_1234",
    InvoiceStatus: "FINALIZED",
    Limit: 20,
})
```
