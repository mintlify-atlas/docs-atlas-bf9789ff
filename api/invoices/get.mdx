---
title: "Get Invoice"
api: "GET /v1/invoices/{id}"
description: "Use when loading an invoice for display or editing (e.g. portal or reconciliation). Supports group_by for usage breakdown and force_runtime_recalculation."
---

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the invoice to retrieve
</ParamField>

## Query Parameters

<ParamField query="group_by" type="array">
  Group usage breakdown by specified fields (e.g., source, feature_id, properties.org_id)
  
  Can be provided multiple times to group by multiple dimensions:
  - `?group_by=source`
  - `?group_by=source&group_by=feature_id`
  - `?group_by=properties.org_id` (for nested properties)
</ParamField>

<ParamField query="force_runtime_recalculation" type="boolean" default="false">
  Force recalculation of usage breakdown at runtime instead of using cached values
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://api.flexprice.io/v1/invoices/inv_1234567890" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash cURL (with usage breakdown)
curl -X GET "https://api.flexprice.io/v1/invoices/inv_1234567890?group_by=source&group_by=feature_id" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```python Python
client = Flexprice(api_key="YOUR_API_KEY")

invoice = client.invoices.get("inv_1234567890")

# With usage breakdown
invoice = client.invoices.get(
    "inv_1234567890",
    group_by=["source", "feature_id"]
)
```

```typescript TypeScript
const client = new Flexprice({ apiKey: "YOUR_API_KEY" });

const invoice = await client.invoices.get("inv_1234567890");

// With usage breakdown
const invoice = await client.invoices.get("inv_1234567890", {
  groupBy: ["source", "feature_id"]
});
```

```go Go
client := flexprice.NewClient("YOUR_API_KEY")

invoice, err := client.Invoices.Get(context.Background(), "inv_1234567890", nil)

// With usage breakdown
invoice, err := client.Invoices.Get(context.Background(), "inv_1234567890", &flexprice.InvoiceGetParams{
    GroupBy: []string{"source", "feature_id"},
})
```
