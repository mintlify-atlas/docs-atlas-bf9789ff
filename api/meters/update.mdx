---
title: "Update Meter"
api: "PUT /v1/meters/{id}"
description: "Update a meter's filters. Other properties are immutable after creation."
---

## Path Parameters

<ParamField path="id" type="string" required>
  Unique identifier of the meter to update.
  
  **Example:** `"meter_550e8400e29b41d4a716446655440000"`
</ParamField>

## Request Body

<ParamField body="filters" type="array" required>
  Updated array of filters to apply before aggregation. This replaces all existing filters.
  
  <Expandable title="filter properties">
    <ParamField body="filters[].key" type="string" required>
      Property name in event data to filter on.
      
      **Example:** `"region"`
    </ParamField>
    
    <ParamField body="filters[].values" type="array" required>
      Array of values to match (OR condition).
      
      **Example:** `["us-east-1", "us-west-2"]`
    </ParamField>
  </Expandable>
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique identifier for the meter.
</ResponseField>

<ResponseField name="name" type="string">
  Human-readable name for the meter.
</ResponseField>

<ResponseField name="tenant_id" type="string">
  Tenant ID that owns this meter.
</ResponseField>

<ResponseField name="event_name" type="string">
  Name of the event being tracked.
</ResponseField>

<ResponseField name="aggregation" type="object">
  Aggregation configuration.
</ResponseField>

<ResponseField name="filters" type="array">
  Updated array of filters.
</ResponseField>

<ResponseField name="reset_usage" type="string">
  Usage reset configuration.
</ResponseField>

<ResponseField name="status" type="string">
  Status of the meter.
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp when the meter was created.
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 timestamp when the meter was last updated.
</ResponseField>

## Example Request

```bash cURL
curl --request PUT \
  --url https://api.flexprice.io/v1/meters/meter_550e8400e29b41d4a716446655440000 \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --data '{
    "filters": [
      {
        "key": "region",
        "values": ["us-east-1", "eu-west-1"]
      },
      {
        "key": "environment",
        "values": ["production"]
      }
    ]
  }'
```
