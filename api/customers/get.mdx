---
title: "Get Customer"
api: "GET /v1/customers/{id}"
description: "Retrieve a single customer by ID"
---

Use this endpoint when you need to load a single customer (e.g., for a billing portal or to attach a subscription).

## Path Parameters

<ParamField path="id" type="string" required>
  The unique FlexPrice customer ID (e.g., `cust_abc123xyz789`).
</ParamField>

## Query Parameters

<ParamField query="expand" type="string">
  Expand related resources in the response (e.g., `parent_customer` to include parent customer details).
</ParamField>

## Response

<ResponseField name="id" type="string">
  The unique FlexPrice customer ID.
</ResponseField>

<ResponseField name="external_id" type="string">
  The external identifier from your system.
</ResponseField>

<ResponseField name="name" type="string">
  The customer's name or company name.
</ResponseField>

<ResponseField name="email" type="string">
  The customer's email address.
</ResponseField>

<ResponseField name="address_line1" type="string">
  Primary address line.
</ResponseField>

<ResponseField name="address_line2" type="string">
  Secondary address line.
</ResponseField>

<ResponseField name="address_city" type="string">
  City name.
</ResponseField>

<ResponseField name="address_state" type="string">
  State or region name.
</ResponseField>

<ResponseField name="address_postal_code" type="string">
  Postal code.
</ResponseField>

<ResponseField name="address_country" type="string">
  Two-letter ISO 3166-1 alpha-2 country code.
</ResponseField>

<ResponseField name="parent_customer_id" type="string">
  The parent customer ID if this is a child customer.
</ResponseField>

<ResponseField name="metadata" type="object">
  Custom metadata key-value pairs.
</ResponseField>

<ResponseField name="environment_id" type="string">
  The environment ID this customer belongs to.
</ResponseField>

<ResponseField name="tenant_id" type="string">
  The tenant ID this customer belongs to.
</ResponseField>

<ResponseField name="status" type="string">
  The status of the customer (e.g., `published`, `draft`, `archived`).
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of when the customer was created.
</ResponseField>

<ResponseField name="updated_at" type="string">
  ISO 8601 timestamp of when the customer was last updated.
</ResponseField>

<ResponseField name="created_by" type="string">
  The ID of the user who created the customer.
</ResponseField>

<ResponseField name="updated_by" type="string">
  The ID of the user who last updated the customer.
</ResponseField>

<ResponseField name="parent_customer" type="object">
  The parent customer object if `expand=parent_customer` is specified and a parent exists.
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET https://api.flexprice.io/v1/customers/cust_abc123xyz789 \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash With Expand
curl -X GET "https://api.flexprice.io/v1/customers/cust_abc123xyz789?expand=parent_customer" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```javascript JavaScript
const response = await fetch(
  'https://api.flexprice.io/v1/customers/cust_abc123xyz789',
  {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY'
    }
  }
);

const customer = await response.json();
```

```python Python
import requests

response = requests.get(
    'https://api.flexprice.io/v1/customers/cust_abc123xyz789',
    headers={'Authorization': 'Bearer YOUR_API_KEY'}
)

customer = response.json()
```
</RequestExample>
